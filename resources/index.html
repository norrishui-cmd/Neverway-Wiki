Material Database

Neverway resources guide: materials, locations, drops, and uses

Resource databases support almost every other page: crafting, gifts, quests, combat prep, farming profits, and map routes.

Database columns to add after launch

Use columns for item name, category, location, source, refresh condition, sell value, crafting use, cooking use, gift use, quest use, and whether the item should be saved. This lets one page rank for many long-tail material searches.

Route-based resource advice

Players do not only need item definitions. They need routes: what to collect in one trip, what to ignore, what to store, and when to return home safely. Connect routes to the map and combat pages.
