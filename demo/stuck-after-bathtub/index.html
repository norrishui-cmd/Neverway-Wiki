Neverway Prologue Stuck After Bathtub: Patch 4 Status

Learn the official status of the Neverway Prologue blocker after leaving the apartment bathtub and how to verify, retry, and report the sequence after Patch 4.

Quick answer

Patch 4 says it possibly fixed Fiona getting stuck immediately after leaving the bathtub in the apartment. Install the update before replaying the interaction.

Verified July 18, 2026. This guide separates confirmed facts from unannounced features.

Verify the version

Steam normally updates the Prologue automatically. Restart Steam or inspect downloads before repeating the apartment sequence. An old local build can preserve a blocker that the current patch attempted to address.

Retry carefully

Load the most recent usable save, complete the bathtub interaction normally, and avoid changing window size or rapidly switching inputs during the exit. Record whether Fiona cannot move, the scene fails to advance, or the application freezes.

If it still happens

Report patch date, operating system, input device, save origin, actions before entering the bathtub, and the exact stuck state. Do not promise a universal workaround because the official note only says the issue was possibly fixed.
