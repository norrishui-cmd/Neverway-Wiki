How to Use IJKL Movement Controls in Neverway Prologue

Use IJKL for movement in Neverway Prologue after Patch 4, find the rebinding menu, and diagnose bindings that do not save or display correctly.

Quick answer

Patch 4 added support for rebinding IJKL as movement keys. Open Options, Gameplay, Controls and assign each direction individually.

Verified July 18, 2026. This guide separates confirmed facts from unannounced features.

Assign the keys

From the main menu open Options, choose Gameplay, and enter Controls. Select movement up, left, down, and right, then bind I, J, K, and L. Apply changes and reopen the menu to verify them.

Test the layout

Check movement during normal exploration and the simulation. Bind attack, dodge, interaction, confirm, and cancel around the same hand position if desired, while keeping every required menu action accessible.

Troubleshooting

If a key is rejected, confirm Patch 4 is installed and remove conflicting duplicate bindings. Disconnect unused controllers if prompts switch unexpectedly. Report the language, keyboard layout, and exact action when a binding fails.
