Neverway Prologue Patch 3 Notes: DirectX, Mouse, and Bug Fixes

Neverway Prologue Patch 3 updated graphics libraries, documented Vulkan and D3D11 launch fixes, and repaired duplication, dates, mouse, enemies, and localization.

Quick answer

Patch 3 was published April 13, 2026. Its main technical focus was DirectX 12 boot crashing, with updated graphics libraries and Vulkan or D3D11 launch options.

Verified July 18, 2026. This guide separates confirmed facts from unannounced features.

Startup changes

The developers updated graphics libraries to remediate DirectX 12 boot crashes. They instructed affected players to test /gldevice:Vulkan or /gldevice:D3D11 through Steam Properties and noted Windows 11 Smart App Control as a possible blocker.

Gameplay fixes

The patch removed an item duplication method involving pause and chest opening, corrected email dates after reaching the island, stopped enemies near the bridge from respawning, and blocked unintended Lancelot access through virtual rendering.

Input and language

When controls rely on the mouse, the game now keeps the pointer inside the window. Patch 3 also fixed Simplified Chinese and Spanish strings. Remaining problems should be reported with platform and input details.
