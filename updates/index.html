Neverway Prologue Patch Notes and Update History

Track Neverway Prologue patches for DirectX crashes, Vulkan and D3D11 options, IJKL rebinding, macOS reporting, progression fixes, visuals, and localization.

Quick answer

The latest prominently published Prologue update is Patch 4 from April 20, 2026. It followed Patch 3’s launch work with input, macOS, visual, progression, audio, and localization changes.

Verified July 18, 2026. This guide separates confirmed facts from unannounced features.

Patch 4 highlights

Patch 4 added IJKL movement rebinding, improved macOS crash reporting, and attempted fixes for a small-window snap-resolution crash, occasional black palette color, and getting stuck after leaving the apartment bathtub. It also corrected Fiona sitting on the simulation bench and music pacing near the fishing area.

Patch 3 highlights

Patch 3 updated graphics libraries in response to DirectX 12 boot crashes. The developers published Vulkan and D3D11 Steam launch options and noted Windows 11 Smart App Control as another possible blocker. It also fixed item duplication, email dates, mouse confinement, enemy respawning, and localization strings.

Using the tracker

Check Steam news before applying an older workaround because the Prologue updates automatically. A useful bug report includes operating system, GPU, patch date, location, input method, and whether Vulkan or D3D11 was forced.
