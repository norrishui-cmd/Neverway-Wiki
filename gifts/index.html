Neverway gift guide: favorite gifts, birthdays, and NPC likes

Gift pages become repeat-visit tools when they are fast, filterable, and honest about verified versus unverified information.

Gift table structure

The launch version should include NPC name, loved gifts, liked gifts, neutral gifts, disliked gifts, birthday, source, and patch version. Keep "unknown" values visible so users know the page is being updated rather than abandoned.

Gift strategy

Early players should save rare items until preferences are confirmed. If Neverway includes birthdays or event windows, the gift guide should prioritize limited-time gifts and easy daily relationship gains.
