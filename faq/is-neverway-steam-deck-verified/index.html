What is confirmed

The answer above reflects the currently available official information. It is written for a specific search intent so players can distinguish the full game from the free Prologue and separate confirmed platforms or features from requests and rumors.

Important boundary

Neverway is still in development. A missing announcement is not proof that a feature will never exist, but it also cannot be presented as confirmed. Exact prices, dates, final balance values, complete character schedules, recipes, gifts, and platform parity require a new official source or direct final-build verification.

How to verify the latest status

Check the date and the platform before relying on the answer. Compare the official Neverway site with the current Steam page, install the newest Prologue patch when troubleshooting, and treat community posts as supporting context rather than final confirmation.
