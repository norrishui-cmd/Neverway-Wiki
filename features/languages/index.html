Neverway Languages: English, Japanese, Chinese, Spanish, Portuguese, and Russian

Check Neverway’s currently listed Steam languages, subtitle and interface status, and which Patch 3 and Patch 4 localization changes affected the Prologue.

Quick answer

Steam currently lists interface and subtitle support for English, Japanese, Brazilian Portuguese, Simplified Chinese, Spanish from Spain, and Russian. Full audio is not marked.

Verified July 18, 2026. This guide separates confirmed facts from unannounced features.

Steam language table

The current store listing shows six supported languages for interface and subtitles. Store data can change before October, so final PC and Switch language coverage should be checked separately at launch.

Prologue patches

Patch 3 corrected Simplified Chinese and Spanish localization strings. Patch 4 addressed additional Spanish and Russian feedback. These fixes describe the Prologue build rather than proving identical text or patch timing on Switch.

What is not listed

Steam does not mark full audio for the six languages in the current listing. Other requested languages, regional variants, and platform parity remain unsupported as claims until a storefront or developer announcement adds them.

Frequently asked questions

How many Steam languages are listed?

Six.

Is Simplified Chinese supported?

Yes, for interface and subtitles.

Is full voice acting marked?

No.
