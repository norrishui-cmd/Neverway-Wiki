What it means for players

Use this update as a versioned status check, not as a substitute for the final game. Store fields, release plans, supported platforms, and Prologue fixes can change before launch. The linked hub keeps the practical guide separate from this dated news record.

Before acting on this information, confirm the date, platform, and game version. For Prologue issues, install the newest Steam update first. For release and feature status, compare the official Neverway site with the current Steam listing.
