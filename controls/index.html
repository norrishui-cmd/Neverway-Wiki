Neverway Controls: Keyboard, Mouse, Controller, and Rebinding

Configure Neverway Prologue keyboard, mouse, and controller inputs, remap controls, use IJKL movement, and troubleshoot reversed confirm or cancel buttons.

Quick answer

Neverway Prologue supports customizable input. Open Options, Gameplay, then Controls to rebind keyboard, controller, or mouse buttons; Patch 4 added IJKL movement rebinding.

Verified July 18, 2026. This guide separates confirmed facts from unannounced features.

Change bindings

Return to the main menu, open Options, choose Gameplay, and enter Controls. Select the action and press the replacement input. Verify movement, interaction, attack, dodge, confirm, cancel, and menu navigation before resuming the story.

Keyboard and mouse

Steam describes combat as optimized for keyboard or controller. The official FAQ confirms mouse-button binding, while Patch 3 keeps the pointer inside the game window when a control uses it. Patch 4 specifically added support for rebinding IJKL as movement keys.

Controller troubleshooting

If confirm and cancel feel reversed on an Xbox-style layout, remap them in the game first. If prompts do not match the device, inspect Steam Input and disconnect unused controllers. PC controller support does not confirm an Xbox console release.

Frequently asked questions

Can controls be changed?

Yes, under Options → Gameplay → Controls.

Can mouse buttons be bound?

Yes.

Can I use IJKL?

Yes, after Patch 4.
