Neverway controls: keyboard, controller, mouse binding, and launch fixes

Controls are a real pre-release search need because the official FAQ mentions custom input, mouse binding, keyboard play, controller play, and Windows launch workarounds.

Supported input questions

Neverway is described as optimized for keyboard and controller. Players should check movement, tool selection, menu navigation, fishing inputs, combat dodge or attack buttons, and whether remapping makes repeated farm actions comfortable.

Custom controls and mouse binding

The official FAQ says controls can be customized from the options menu and mouse binding is supported. After launch, this page should list which actions can be rebound, whether controller prompts update correctly, and whether any actions are locked.

Launch option troubleshooting

For Windows launch problems in the prologue, official troubleshooting mentions Steam launch options -force-vulkan, -force-d3d12, and -force-d3d11. This page should keep those commands visible but clearly label them as troubleshooting options, not performance recommendations.
